# ANALYSE-SQL-SUR-GOOGLE-BIGQUERY-Dataset-Iowa-Liquor-Sales
Analyse SQL du dataset Iowa Liquor Sales sur Google BigQuery pour étudier le marché des spiritueux dans l’Iowa. Le projet explore les tendances de ventes, la saisonnalité, la géographie du marché et le positionnement concurrentiel de Pernod Ricard USA à partir de requêtes analytiques SQL.
